# VideoHead

## Guillermo Huerta - 50 Greatest Movies Ever Selection Web App

Web App created on July '20 as part of Aertec Solutions' recruitment process technical test.

Data Source: [The Movie Database (TMDb)](https://www.themoviedb.org/) API (v3).

Built with [Gatsby](https://www.gatsbyjs.org). Deployed to [Netlify](https://www.netlify.com).

Check running application [here](https://videohead.netlify.app).
